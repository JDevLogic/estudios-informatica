# 📓 Apuntes Día 4 - Python (21 de marzo de 2025)

## 📌 ¿Qué son las funciones?
* Son bloques de código que se pueden reutilizar.
* Ayudan a organizar el programa y evitar repetir código.
* Se les puede pasar información (parámetros) y pueden devolver un resultado (return).

---

## 📌 Ventajas de usar funciones
- Reutilización de código.
- Código más limpio y ordenado.
- Se pueden dividir tareas en pequeñas partes.

---

## 📌 Cómo crear una función
* La palabra clave para crear una función es `def`.
* Se le da un nombre y se pueden definir parámetros entre paréntesis.

**Ejemplo básico:**
- Se crea la función llamada `saludar`.
- Dentro, hace una acción: saludar.

---

## 📌 Parámetros y argumentos
* Parámetro → La variable que se pone en la función.
* Argumento → El valor que se le pasa al llamar la función.

| Concepto    | Definición                                             |
|------------|--------------------------------------------------------|
| Parámetro  | Variable que se define en la función                    |
| Argumento  | Valor que se pasa a la función cuando se llama          |

---

## 📌 Uso de `return`
* `return` sirve para devolver un resultado.
* Lo que devuelve se puede guardar en una variable y usarlo después.

---

## 📌 Input dentro de funciones
* Se puede pedir al usuario que escriba algo con `input()`.
* Luego usar esa información dentro de la función o guardarla.
