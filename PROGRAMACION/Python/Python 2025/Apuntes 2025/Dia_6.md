## 🧠 Conceptos clave reforzados:

* `while` se usa cuando **no sabemos cuántas veces** se repetirá el bucle.
* `break` permite **salir del bucle** en cuanto se cumple la condición deseada.
* Con `input()` se puede **hacer que el usuario interactúe** con el programa.
* Usar contadores (`intentos += 1`) es útil para **medir repeticiones o errores**.
* Combinar `if`, `elif`, `else` dentro de bucles nos permite crear **lógica dinámica**.
